# ArogyaNidhi

ArogyaNidhi is an Android application developed to support digital healthcare and medical fund management solutions. The app aims to provide users with an easy and efficient platform for managing healthcare-related information and services.

## Features

- User-friendly Android interface
- Healthcare and fund management support
- Secure data handling
- Fast and responsive performance
- Modern Android development architecture

## Technologies Used

- Java
- Android Studio
- Firebase
- XML
- Gradle

## Project Structure

- `app/` - Main Android application source code
- `gradle/` - Gradle configuration files
- `build.gradle` - Project build configuration
- `settings.gradle` - Project settings
